Regras de Negócio

01.01 = Se o sistema localizar ao menos uma informação do hospital selecionado, apresentará o nome do hospital e a(s) informação(ões) localizadas na tela. Caso contrário, apresentará somente o nome do hospital e emitirá a MSG 01.01 e MSG 01.02. Para este caso as informações do hospital são: logradouro, número, bairro, cidade, Estado e telefone(s).

01.02 = O sistema verificará as coordenadas geográficas do hospital selecionado. As coordenadas geográficas são representadas por 02 atributos (Latitude e Longitude). Se o sistema não localizar um dos atributos ou um deles não estiver no formato estabelecido, o sistema emitirá as mensagens MSG 01.03 e MSG 01.04. As coordenadas geográficas deverão ser do tipo double, com 5 casas decimais. O atributo Latitude deverá aparecer antes do atributo Longitude e deverá estar no intervalo de -90 a 90. E o atributo Longitude, de -180 a 180.


02.01 = Se o usuário não preencher a data, então o sistema não deverá salvar os dados e emitirá a mensagem MSG 02.01.

02.02 = Se a soma dos valores dos componentes não for 100%, então o valor de total deverá ser grifado em vermelho e o sistema não salvará os dados, emitindo a mensagem MSG 02.02.

02.03 = Se um valor ultrapassar 100%, então o sistema não deverá salvar os dados e emitirá a mensagem MSG 02.03.

02.04 = Se a consulta não retornou dados, então o sistema emitirá a mensagem MSG 02.04.

02.05 = Se o usuário desejar excluir um registro, então o sistema emitirá a mensagem MSG 02.05.

02.06 = Se o cadastro foi salvo com sucesso, então o sistema emitirá a mensagem MSG 02.06.

02.07 = Se a exclusão de um registro foi realizada com sucesso, então o sistema emitirá a mensagem MSG 02.07.

02.08 = Se a data preenchida for superior a data atual, então o sistema não deverá salvar os dados e emitirá a mensagem MSG 02.08.

02.09 = Formato de entrada de data é "dd/mm/aaaa".

02.10 = Os valores dos compostos devem pertencer ao conjunto dos números reais positivos.

02.11 = Um valor de compostos deve ter no máximo duas casas decimais.

02.12 = Os compostos a serem trabalhados serão os seguintes: adiposo, tecido mole não músculo-esquelético, músculo esquelético e osso (modelo 4C anatômico).


03.01 = O sistema poderá apenas publicar uma postagem em nome do usuário, sem ter acesso a qualquer outra informaçao de seu perfil.

03.02 = O sistema apresentará uma prévia da mensagem que será publicada no Facebook MSG 03.01.

03.02 = O sistema apresentará uma mensagem de erro MSG 03.02.


04.01 = Se sistema for salvar formulario sem preencher todos os campos emitir mensagem MSG 04.01.

04.02 = Se sistema for salvar formulario sem anexar arquivo emitir mensagem MSG 04.02.

04.03 = Se sistema for fechar formulario e houver campos sem salvar emitir mensagem MSG 04.03.

04.04 = Se sistema for salvar formulario e extensão de arquivo não for .pdf de emitir mensagem MSG 04.04.


05.01 = Se houver algum campo vazio o sistema não salva e exibe mensagem MSG 05.01.

05.02 = Se salvo com sucesso, exibe mensagem MSG 05.02.

05.03 = Se a categoria está sendo utilizada, não permite a exclusão e exibe mensagem MSG 05.03.

05.04 = Se a categoria não está sendo utilizada, exibe mensagem MSG 05.04.

05.05 = Se algum campo possui menos de 3 caracteres, exibir mensagem MSG 05.05.


06.01 = Exames: urina, sangue, cardiológico e cateterismo.

06.02 = O usuário deverá clicar no botão Gera PDF para gerar arquivo.

06.03 = Se não houver histórico de exame, exibe mensagem MSG 06.01 e não gera arquivo.

06.04 = Caso o usuário não informe filtro de busca, gera pdf com todos exames cadastrados;Caso contrário, exibe histórico de acordo com filtro.
